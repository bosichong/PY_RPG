# PY_RPG

### 介绍

PY_RPG是对PYGAME游戏库在实际应用中的一个简单封装，使游戏的设计采用面向对象方式。

再次封装了大量的常用的方法，目前还在完善中，希望有兴趣的朋友，一起来维护哈。
